# Plattinger.no

Nettsted for **Plattinger AS** - spesialister på betongrenovering og overflatebehandling.

## 🏗️ Om prosjektet

Dette er en moderne, responsiv nettside bygget med:
- **HTML5** - Semantisk struktur
- **CSS3** - Responsivt design med Flexbox/Grid
- **JavaScript (ES6+)** - Interaktivitet og skjema-validering
- **Vite** - Rask build-tid og utviklingsserver

## 🚀 Deploy

Nettsiden er deployet på **Netlify** med automatisk deploy ved push til main-branchen.

**Live URL:** https://plattinger.no

## 📁 Prosjektstruktur

```
plattinger-no/
├── src/
│   ├── index.html      # Hovedside
│   ├── styles.css      # Styling
│   └── app.js          # JavaScript logikk
├── package.json        # Node.js avhengigheter
├── netlify.toml        # Netlify konfigurasjon
└── README.md           # Denne filen
```

## 🛠️ Lokal utvikling

```bash
# Installer avhengigheter
npm install

# Start utviklingsserver
npm run dev

# Bygg for produksjon
npm run build
```
